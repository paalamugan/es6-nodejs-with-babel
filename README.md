```shell
npm install
```

## Get Started

- To development mode

```shell
npm run dev
```

- To production mode

```shell
npm run prod
```

- To build a server code es6 to es5

```shell
npm run build
```

- Watching files changes

```shell
npm run watch
```

### Other scripts

- transpile - convert es6 and beyond code to es5 to a folder named `complied-server`
- clean - delete transpiled folder
- build - clean and transpile
